# Acunetix XML parser

### Requirements
This script requires: 
* openpyxl (https://pypi.python.org/pypi/openpyxl)
* Python 3.x

Install
You can install libnmap via pip:

pip install openpyxl

### Usage
```
python acunetix-parser.py <Directory with XML Files> <XLSX output file>
```
